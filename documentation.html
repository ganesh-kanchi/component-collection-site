Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials or placeholder icon. It can be used in login page, notification bar, and in various parts of the project etc

Circular Avatar

Circular Avatars can be created using the respective avatar class inside the image tag.

Square Avatar

Square Avatars can be created using the respective avatar class inside the image tag.

Avatar with placeholder icon

Avatar with placeholder icon can be created using the class avatar-placeholder, inside the img tag.

Avatar with Placelolder text

Avatar with placeholder initial can be created using the respective avatar-placeholder-text class, inside the image component tag. And the class flex-container will be used to center the initial.

Badge

Badges are used to highlight an item's status for quick recognition. It is generally used to show the count of notification or status of the user.

Status Badge

Status badge is used to show the status of the user. It has 3 variants online-badge | dnd-badge | idle-badge

Icon badge

Icon badge is used to show the count of the notification or add cart option. It can be created using the class badge-icons icon-badge Font size of badge text is in em. It will change as per the parent element's font-size.

Button

The Button component is used to trigger an action or event, such as submitting a form, opening a dialog, cancelling an action, or performing a delete operation.

Contained button

Contained buttons are high-emphasis, distinguished by their use of elevation and fill.

Outlined button

Outlined buttons are generally used in secondary purposes where more emphasis is not needed.

Link Button

Link buttons are used to add the link in the button. Use the class link-button in the button

Icon Button

Icon button can be created using the class icon-button and add the appropriate icon.
